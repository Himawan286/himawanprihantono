Tentang Saya

Selamat datang! Nama saya Himawan Prihantono, seorang pengembang web yang bersemangat dengan lebih dari 2 tahun pengalaman dalam dunia pemrograman. Sejak pertama kali mengenal dunia digital, saya telah terpikat oleh kemampuan teknologi untuk menciptakan solusi yang inovatif dan memecahkan masalah nyata.

Saya memulai perjalanan saya dengan belajar dasar-dasar pemrograman, dan dengan cepat mengembangkan keterampilan dalam berbagai bahasa pemrograman seperti HTML, CSS, JavaScript, dan PHP. Saya juga telah bereksperimen dengan berbagai framework dan alat, seperti React dan Bootstrap, yang memungkinkan saya untuk membangun aplikasi web yang responsif dan menarik.

Saat ini, saya melanjutkan studi di Politeknik UNJ dalam program D4, di mana saya terus memperdalam pemahaman saya tentang pengembangan perangkat lunak, algoritma, dan struktur data. Saya percaya bahwa pendidikan formal dan pengalaman praktis saling melengkapi, sehingga saya aktif berpartisipasi dalam proyek-proyek kelompok dan hackathon untuk meningkatkan keterampilan kolaborasi dan komunikasi saya.

Visi saya adalah untuk menciptakan teknologi yang berdampak positif pada masyarakat. Saya berkomitmen untuk terus belajar dan beradaptasi dengan perkembangan terbaru di dunia teknologi. Selain itu, saya juga ingin berbagi pengetahuan saya dengan orang lain, karena saya percaya bahwa kolaborasi dan berbagi informasi adalah kunci untuk kemajuan bersama.

Di luar pemrograman, saya memiliki minat dalam desain grafis dan fotografi, yang membantu saya mengembangkan perspektif kreatif dalam proyek pengembangan web saya. Saya berharap dapat membangun jaringan dengan para profesional lain di industri ini dan bersama-sama menciptakan solusi inovatif yang membawa perubahan.
